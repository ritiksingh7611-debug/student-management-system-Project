# student-management-system-Project
simple student management system in c++
Student Management System in C++
------------------------------------

Introduction
------------

This project is a Student Management System developed using C++. It is a simple console-based application that allows users to store and manage student information such as roll number, name, and marks. The project demonstrates the basic concepts of object-oriented programming in a clear and practical way.

Objectives
------------

To store student records efficiently
To display all student details
To search a student using roll number
To understand object-oriented programming concepts
To improve logical and coding skills

Methodology / System Design
---------------------------

The system is designed using a class named Student.

Each student contains:

roll number
name
marks

Working of the system:
--------------------------

The program uses a menu-driven approach
User selects an option from the menu
Data is stored using an array of objects
Records can be displayed or searched

Concepts used:
-----------------

class and object
encapsulation (private data members)
arrays
loops and conditional statements

Implementation
--------------

The project is implemented in C++ using a structured and modular approach.

Main functionalities:

Add Student
Display All Students
Search Student by Roll Number

The program uses:

class functions for input and display
loop for managing multiple records
switch-case for menu operations


Conclusion
----------

This project helps in understanding the core concepts of C++ such as classes, objects, and data handling. It is simple, user-friendly, and suitable for beginners. The project can be further improved by adding file handling and more advanced features.
